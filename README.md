# SDC Infrastructure Automation - PBL Semester 4

A static HTML presentation and portfolio built for Project Based Learning (PBL).

**Under the guidance of Dr. Dibakar Sinha**

## Structure
- `index.html`: primary presentation entrypoint
- `index_simple.html`: alternate simplified version
- `public/`: static image assets

## Features
- Static single-file presentation flow
- Tailwind CDN styling with custom UI and projector-oriented layout
- Mermaid and Chart.js powered diagrams and charts
- GitHub Pages friendly deployment with no build step

## Local Usage
Open `index.html` directly in a browser, or serve the repository as static files with any simple HTTP server.

## Deployment
Deploy the repository as static files. No Slidev, Vue build, or npm pipeline is required.
